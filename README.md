# DeciDo

[![Status](https://img.shields.io/badge/Status-In%20Development-orange.svg)](https://gitflic.ru/project/wirrwarr/decido)

A mobile application designed to simplify and enhance decision-making processes for groups and individuals

## Features

- Registration, authorization
- Social connections module: adding friends, creating groups
- Creating and editing lists
- Randomizer mode: Random selection from a list.
- Voting mode (Ranking): Users rank items from a list in order (from most to least preferred, no equal ratings for different items), where the number of places equals the number of items. Finally, when all participants are ready, the votes are automatically counted and the result is displayed.

## About the project
**Problems solved by the app:**
1. **Difficulty of choice for a single user:** When a user has a list of options, for example, what they'd like to watch or play, but all the options seem equally appealing, they spend hours choosing instead of simply starting to watch or play;
2. **Conflicts of interest in a couple/group:** When choosing a leisure activity for several people, arguments often arise, and it's difficult to reach an agreement, as everyone has different interests.

**Project goals:** Create a tool that makes decision-making fast, objective, and engaging.

**Project objectives:**
1. Define application requirements;
2. Design the architecture;
3. Implement the mobile app according to the technical specifications;
4. Test and conduct a trial run.
