# DeciDo

[![Status](https://img.shields.io/badge/Status-In%20Development-orange.svg)](https://gitflic.ru/project/wirrwarr/decido)

A mobile application designed to simplify and enhance decision-making processes for groups and individuals

## Features

- Registration, authorization
- Social connections module: adding friends, creating groups
- Creating and editing lists
- Randomizer mode: Random selection from a list.
- Voting mode (Ranking): Users rank items from a list in order (from most to least preferred, no equal ratings for different items), where the number of places equals the number of items. Finally, when all participants are ready, the votes are automatically counted and the result is displayed.
